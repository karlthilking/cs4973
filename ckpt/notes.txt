Replacing library calls and system calls with wrapper functions
- use LD_PRELOAD to load dynamically linked shared library (the checkpoint 
  library into the executable to be checkpointed), then calls to library
  or system call routines will enter into the wrapper first
- use syscall(SYS_xxx, ...) to call system calls directly
- use dlysm to get real library function call into libc after wrapper
  does its works (use RTLD_NEXT)
